☁ Cloud-Sync einrichten

Deine Kurse werden in einem GitHub Gist gespeichert — kostenlos, ohne Server, von jedem Gerät abrufbar.

Einmalige Einrichtung:
  1. Gehe zu github.com/settings/tokens/new
  2. Name: z.B. Kursorganisator
  3. Wähle nur gist als Berechtigung
  4. Token generieren und hier einfügen
  5. Gist-ID leer lassen → wird automatisch angelegt
